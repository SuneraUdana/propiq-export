# PropIQ — Property Investment Scoring Engine

AI-powered property investment analysis for Australian suburbs.

## Features
- 🏘️ Suburb scoring and ranking
- 🤖 Grounded AI chat (Groq LLaMA 3.3 70B)
- 📊 Outcome tracking with hit rate
- 🧠 Episodic memory (pipeline + conversation history)

## Setup
Set `GROQ_API_KEY` as a Space Secret in Settings → Variables and secrets.
